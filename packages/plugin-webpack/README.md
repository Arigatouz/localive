# @localive/webpack

> Webpack dev-server plugin for Localive — saves live i18n edits back to your locale files.

Part of [Localive](https://github.com/localive/localive) — live, in-context i18n editing for React, Vue, Angular, and Svelte.

## Install

```sh
npm install -D @localive/webpack
```

Requires the peer dependencies `webpack` (>=5) and `@localive/vite`.

See the full documentation at **https://localive.dev**.
