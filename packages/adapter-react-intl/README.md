# @localive/adapter-react-intl

> react-intl adapter for Localive — bridges react-intl to Localive's live i18n editing.

Part of [Localive](https://github.com/localive/localive) — live, in-context i18n editing for React, Vue, Angular, and Svelte.

## Install

```sh
npm install @localive/adapter-react-intl
```

Requires `@localive/core` and `react-intl` in your app.

## Usage

See the full documentation at **https://localive.dev**.
